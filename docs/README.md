# 기능 목록
- [x] Number가 있다.
  - [x] Number는 1~9 사이의 값이 아니면 예외가 발생한다.
  - [x] Number는 동등성을 보장한다.
- [x] Position이 있다.
  - [x] Position은 1~3 사이의 값이 아니면 예외가 발생한다.
  - [x] Position은 동등성을 보장한다.
- [x] Ball은 Position과 Number로 구성된다.
- [ ] Referee가 있다.
  - [ ] Referee는 두개의 Ball을 판정한다.
- [ ] Balls는 3개의 Ball로 구성된다.
